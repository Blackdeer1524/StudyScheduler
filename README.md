# StudyScheduler

Равномерно распределяет лекции по дням до дедлайна с учётом реального времени каждой лекции.

## Как работает

1. Вы описываете параметры в `subjects.yaml`: даты и оставшиеся лекции по каждому предмету.
2. `planner.py` вычисляет дневной бюджет (`суммарные минуты / дней до дедлайна`) и жадно раскладывает лекции по дням строго до дедлайна, балансируя прогресс по предметам.
3. `generate.py` преобразует текстовый план в `.ics`-файл для импорта в календарь.

## Установка

```bash
pip install -r requirements.txt
```

## Формат subjects.yaml

```yaml
config:
  start_date: 2026-03-18   # первый день расписания
  deadline: 2026-04-15     # лекции планируются СТРОГО ДО этой даты

subjects:
  "Линейная алгебра":
    5: {hours: 1, minutes: 25}
    6: {hours: 1, minutes: 36}

  "Анализ":
    12: {hours: 1, minutes: 20}
    13: {hours: 1, minutes: 37}
```

- В `subjects` перечисляются только **ещё не просмотренные** лекции.
- Ключ каждой лекции — её порядковый номер; лекции внутри предмета всегда идут по порядку.

## Запуск

### С Makefile

```bash
# Сгенерировать текстовый план и .ics одной командой
make generate
```

| Цель | Что делает |
|---|---|
| `make calendar.txt` | Запускает `planner.py`, фильтрует строки `FREE DAY`, сохраняет в `calendar.txt` |
| `make generate` | Запускает `generate.py` поверх `calendar.txt`, пишет `custom_study_calendar.ics` |

### Без Makefile

```bash
# Шаг 1 — текстовый план
python planner.py | grep -v FREE > calendar.txt

# Шаг 2 — .ics-файл
python generate.py calendar.txt > custom_study_calendar.ics
```

### Только просмотр плана в терминале

```bash
python planner.py
```

Пример вывода:

```
[2026-03-18]:Анализ 7 (3h 24m)
[2026-03-19]:Линейная алгебра 2 (1h 55m)
[2026-03-19]:Теория вероятностей 10 (1h 25m)
...
[2026-04-14]:Теория вероятностей 29 (1h 27m)
```

Последний день в расписании — `deadline - 1 день`.

## Импорт в календарь

После генерации `custom_study_calendar.ics` импортируйте его в Google Calendar, Apple Calendar или любое другое приложение, поддерживающее формат iCalendar.
